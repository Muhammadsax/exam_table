# Exam Schedule Page / صفحة جدول الامتحانات

A simple, clean web page to display a university exam schedule with a feature to download the schedule as a PDF file.

صفحة ويب بسيطة ونظيفة لعرض جدول امتحانات جامعي مع ميزة لتحميل الجدول كملف PDF.

---

## Features / الميزات

- **Clear Layout:** Displays the exam schedule in an easy-to-read table format.
- **PDF Export:** A "Download as PDF" button to save the schedule for offline viewing or printing.
- **Responsive Design:** The layout is adapted for different screen sizes.

---

- **تنسيق واضح:** عرض جدول الامتحانات في جدول منظم وسهل القراءة.
- **تصدير PDF:** زر "تحميل كـ PDF" لحفظ الجدول للعرض دون اتصال بالإنترنت أو للطباعة.
- **تصميم متجاوب:** تم تكييف التنسيق ليتناسب مع مختلف أحجام الشاشات.

---

## How to Use / كيفية الاستخدام

1.  **Open the page:** Simply open the `index.html` file in any modern web browser.
2.  **Download PDF:** Click the "Download as PDF" button at the top to generate and save the schedule.

---

1.  **فتح الصفحة:** ببساطة، افتح ملف `index.html` في أي متصفح ويب حديث.
2.  **تحميل PDF:** انقر على زر "تحميل كـ PDF" الموجود في الأعلى لإنشاء وحفظ الجدول.

---

## Technologies Used / التقنيات المستخدمة

- **HTML5**
- **CSS3**
- **JavaScript**
- **html2pdf.js** - For PDF generation.
